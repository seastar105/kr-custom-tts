# kr-custom-tts

kr-custom-tts는 머신러닝을 잘 모르는 사람도 문장을 읽어주는 한국어 TTS를 만들기 위한 프로젝트입니다.


```
org
├── 00000.wav
├── 00001.wav
├── 00002.wav
├── 00003.wav
├── 00004.wav
├── 00005.wav
├── 00006.wav
├── 00007.wav
..
..
..
├── 00290.wav
├── 00291.wav
├── 00292.wav
├── 00293.wav
├── 00294.wav
├── 00295.wav
├── spk_id
└── transcripts.txt
```

사용하는 라이브러리

- [ESPNet](https://github.com/espnet/espnet)
- [ffmpeg-normalize](https://github.com/slhck/ffmpeg-normalize)
- [g2pk](https://github.com/Kyubyong/g2pK)
